This sample was built using the chat playground experience and published to a web app in Azure
The chat playground config is in the same folder as this document.
GPT4 model is used in the Web App (https://sowassistant.azurewebsites.net/), since it suited this use case better than the turbo model.
In this example, a user can create a new SOW Document based on a template, look up other SOW Documents created earlier to repurpose content when creating the new document.
Azure Cognitive Search is used for searching other SOW Documents
